# 首尾帧衔接工具 - 使用说明

## 安装依赖

在使用工具之前，请先安装所需的依赖包：

```bash
pip install -r requirements.txt
```

## 配置 API Key

1. 打开 `.env` 文件
2. 根据你要使用的模型，填写对应的 API Key
3. 保存文件

例如，如果你要使用豆包模型：
```
DOUBAO_API_KEY=你的_api_key
```

## 使用步骤

### 1. 选择图片
- 点击"选择起始"按钮，选择过渡开始的图片
- 点击"选择结束"按钮，选择过渡结束的图片

### 2. 配置 AI 模型
- 从"视觉模型"下拉框选择用于分析图片的模型
- 从"文本模型"下拉框选择用于生成提示词的模型
- 在"API Key"输入框中输入对应的 API Key

### 3. 设置视频参数（可选）
- 分辨率：输出视频的宽高
- 帧率：视频的帧率（fps）
- 时长：过渡视频的时长（秒）
- 过渡：过渡效果类型（morph/flow/blend）
- 缓动：缓动曲线类型

### 4. 输入转换要求
在"转换要求"文本框中描述你希望的视频过渡效果。

### 5. AI 分析并生成提示词
点击"🎯 AI分析并生成"按钮，AI会：
1. 分析两张图片的内容
2. 根据你的要求生成过渡提示词

### 6. 生成视频
- 选择输出目录
- 点击"🚀 生成视频"按钮
- 等待视频生成完成

## 支持的模型

### 文本对文本模型
- 国外：GPT-4o (OpenAI)、Claude 3.5 Sonnet (Anthropic)、Gemini 2.0 Pro (Google)
- 国内：DeepSeek Chat、Qwen Max (通义千问)、Doubao Pro (豆包)、Yi Large (零一万物)、Moonshot (月之暗面)、Spark 4.0 (星火)

### 图像分析模型
- 国外：GPT-4o Vision (OpenAI)、Gemini 2.0 Vision (Google)
- 国内：Qwen VL Max (通义千问)、Doubao Pro Vision (豆包)

### 其他模型类别（预留）
- 文本到图像
- 文本到视频
- 图像到视频
- 视频到视频

## 注意事项

1. 请确保 API Key 正确且有效
2. 生成视频需要一定时间，请耐心等待
3. 建议使用高清图片以获得更好的过渡效果
4. 不同的过渡效果适合不同的场景，可以尝试多种效果
